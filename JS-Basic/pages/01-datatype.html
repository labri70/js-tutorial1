JS-Basic(01) - Data Type

자바스크립트의 null은 '의도적으로 값이 없음'을 명시하기 위한 기본 데이터 타입이다.
타입도 null이며, 값도 null인 Primitive Type이다. 즉, null은 객체가 아닙니다! (기본 타입입니다!)
null은 객체가 아닌 원시 자료형임에도 type을 찍어보면 object로 나오는데, 이는 자바스크립트를 처음 구현할 때 발생한 버그라고 합니다.
→ 이는 typeof를 구현한 코드에 null을 체크하는 항목을 누락해서 발생한 버그입니다.

그럼 null 타입 체크는 어떻게 하면 좋을까?

# null 타입 체크 방법? 일치 연산자 (===)을 쓰는게 좋습니다. 일치 연산자는 값과 타입이 동일할 때에만 true를 반환합니다. 

    
      var null_02 = null;
      null_02 === null;   // true

자바스크립트의 null은 객체가 아닌 기본타입 이며, 타입체크는 일치연산자(===)로 하는게 좋다는 걸 기억해두면 좋겠다.
